1.2. Save that text to /tmp/key, and then import it as follows:

```
gpg --import /tmp/key
```

2.  Download SafePay binary (`$FILENAME`) and signature file (`$FILENAME.sig`) 
3.  Verify the signature by running:

```bash
$ gpg --verify \
 $FILENAME.sig \
 $FILENAME

# It should return:
Good signature from "SafePay (SafeCoin team) <safecoin@safecoin.org>"
```
